1. What is Salesforce Platform?

The Salesforce Platform is a cloud-based platform that allows businesses to build, customize, and manage applications using Salesforce tools and technologies.

It helps users to:

Create custom applications
Store and manage data
Automate business processes
Build reports and dashboards
Customize standard Salesforce features
Develop apps using low-code and coding methods

The platform supports both administrators and developers for creating business solutions.

2. Explain the Following
a) App

An App in Salesforce is a collection of:

Objects
Tabs
Permissions
Features

that work together for a business purpose.

Example:
Sales App
Service App
HR Management App

Apps help users access all required tools in one place.

b) Object

An Object is a database table used to store data in Salesforce.

Objects contain:

Records (rows)
Fields (columns)
Types of Objects:
Standard Objects
Account
Contact
Opportunity
Custom Objects
Student
Project
Employee
Example:

A Student object stores:

Student Name
Roll Number
Branch
Phone Number

c) Tab

A Tab is used to access an object or feature in Salesforce.

Tabs appear on the navigation bar and help users open records quickly.

Example:
Accounts Tab
Contacts Tab
Opportunities Tab

Clicking a tab opens the related object data.

3. Difference Between Configuration and Coding
Configuration	Coding
Uses low-code or no-code tools	Uses programming languages
Easy to implement	Requires developer knowledge
Done using clicks and settings	Done using code
Used for simple automation	Used for complex logic
Managed by Admins	Managed by Developers
Examples: Workflow, Validation Rules	Examples: Apex, Lightning Web Components
Example:
Creating a custom field → Configuration
Writing Apex trigger → Coding
